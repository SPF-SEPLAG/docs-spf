# Dashboard Execução Orçamentária
Esse painel foi desenvolvido para consolidar informações a respeito da Execução Orçamentária da SEPLAG.

---

## Objetivo
Oferecer informações relevantes e atualizadas, a fim de subsidiar a melhoria da tomada de decisão e permitir o melhor acompanhamento da execução orçamentária pelas áreas técnicas.

---

## Público-alvo
- Diretoria de Planejamento e Orçamento (DPO).
- Subsecretários(as) e Secretário(a).
- Áreas responsáveis pelas ações orçamentárias.

---

## Fontes de Dados
| Fonte | Tipo |
|--------|------|
| Planilha DPO | Excel (Sharepoint) |
| Bases de dados tratadas via Python + Github Actions + msal | .csv |

## Atualizações e Manutenção
- Os painéis são atualizados **diariamente**, de **hora em hora**, das 10h-17h.
- Configuração do Modelo Semântico do Power BI online e dos cron jobs do github actions.

---

## Tecnologias e integrações

- **Power BI (Desktop e Online)** — plotagem de visualizações e atualização automática dos dados
- **Python (pandas) / Power Query / DAX** — transformação e modelagem de dados  
- **github actions** — execução automática de scripts para tratamento e e atualização das bases.
