# O que é o Github Actions?

Segundo a [documentação oficial do Github](https://docs.github.com/pt/actions/about-github-actions/understanding-github-actions), “o gitHub fornece máquinas virtuais do Linux, Windows e macOS para executar seus fluxos de trabalho, ou você pode hospedar seus próprios executores auto-hospedados na sua própria infraestrutura de dados ou na nuvem.” 

A partir da configuração de um arquivo de script .yml é possível estipular a execução desses fluxos de trabalho pelas máquinas virtuais disponibilizadas pelo github, de forma automática, definindo inclusive agendamentos de horários para execução (chamados de cron jobs). 

A partir das VMs disponibilizadas pelo Github Actions e pela definição dos cron jobs por meio do arquivo `.yml` se torna possível automação de tarefas através da execução automática de scripts.
